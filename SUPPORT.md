# Support

## 获取帮助

如果你在本地运行或二次开发时遇到问题，建议优先查看以下文档：

- [README.md](README.md)
- [docs/quick-start-windows.md](docs/quick-start-windows.md)
- [docs/troubleshooting.md](docs/troubleshooting.md)
- [docs/release-checklist.md](docs/release-checklist.md)

## 提交问题

如果文档无法解决，请创建 GitHub Issue，并提供：

- 问题描述
- 复现步骤
- 实际结果和期望结果
- 运行环境
- 相关日志或截图

## 安全问题

安全漏洞请参考 [SECURITY.md](SECURITY.md)，不要在公开 Issue 中披露漏洞利用细节。
