# Security Policy

## 支持版本

当前仓库处于早期开发阶段，优先维护 `main` 分支和最新发布版本。

| 版本 | 支持状态 |
| --- | --- |
| main | 支持 |
| 最新 Release | 支持 |
| 旧版本 | 尽力支持 |

## 报告安全问题

如果你发现安全漏洞，请不要在公开 Issue 中披露利用细节。建议通过私有渠道联系项目维护者，或先提交一个不包含敏感细节的 Issue，说明需要私下沟通。

报告时请尽量包含：

- 受影响模块或接口
- 复现步骤
- 风险影响范围
- 已知缓解措施

## 凭据与配置

仓库中的账号密码均为本地开发默认值。部署到真实环境前必须通过环境变量或密钥管理系统替换 `JWT_SECRET`、数据库密码、RabbitMQ 密码和 Grafana 密码。
