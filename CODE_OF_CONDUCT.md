# Code of Conduct

## 我们的承诺

我们希望本项目保持专业、开放和尊重的协作环境。参与者应以建设性的方式讨论问题、提交改动和评审代码。

## 期望行为

- 使用友好、明确、尊重的语言。
- 聚焦事实、代码、设计和问题本身。
- 接受不同经验背景下的合理观点。
- 在发现问题时提供可执行的改进建议。

## 不可接受行为

- 人身攻击、骚扰、歧视或威胁。
- 发布他人的隐私信息。
- 恶意刷屏、误导性报告或故意破坏协作流程。
- 在公开渠道披露未修复安全漏洞的利用细节。

## 执行

维护者可以对违反本准则的行为进行提醒、编辑、关闭讨论、限制参与或移除相关内容。严重或重复违规可能导致永久禁止参与项目。
