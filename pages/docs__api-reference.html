核心接口文档

1. 通用说明

网关地址：

http://127.0.0.1:8080

统一响应格式：

{
  "success": true,
  "code": "OK",
  "message": "success",
  "data": {},
  "timestamp": "2026-05-29T00:00:00+08:00"
}

除 /api/auth/dev-token 和 actuator 相关接口外，其余业务接口建议携带：

Authorization: Bearer <accessToken>

获取开发 token：

curl -X POST "http://127.0.0.1:8080/api/auth/dev-token?username=demo&role=ADMIN"
